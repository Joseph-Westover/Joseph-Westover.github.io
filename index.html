Space X Launch Analysis

Data Collection & Wrangling

I began by scraping web data using BeautifulSoup and converting it into a dataframe with Pandas. Next I queried it with SQL, and began my analysis with Python.

Plotting Launches on US Map

I plotted failed and successful Space X launches on a US map using the Python Folium library. Failed launches are coded as red and successful ones are green. The goal was to see if launch site location influenced the chance of success.

Visualizing the Data with Plots

In order to better understand how variables influenced each other I created charts to visualize the data. In the bar chart shown, I compare mission success (class) vs the orbit the mission was going to. Class = 1.0 had 100% success rate and Class = 0.0 had a 0% success rate.

Cleveland Volleyball Court Finder

Building the DataFrame

I began by creating a dictionary of courts and then converting it into a DataFrame using Pandas.

Finding the Closest Court

Using a distance function, I used a for loop to iterate through the DataFrame and find the distance to each court. Then I found the index of the smallest distance to use for plotting.

Plotting Distance to Courts

With Folium I plotted the location of all courts, the user's location (denoted by a head), and the closest court (denoted by a star). I also added lines to each court from the user's location. If user clicks on a marker I have programmed the name of the court and distance to show.
